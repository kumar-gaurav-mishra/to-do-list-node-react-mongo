# to-do-list-node-react-mongo
This project contains server and client side complet code for to do list.
Client will be coded on react and context api
server will be coded on Node mongo and express with authentication.
 #To use the project
 1.) go to client and server folders and do npm install
 2.) Make sure you have node, mongo installed
 3.) to run the project in main folder just do npm start
